# Frontend (Next.js 16)

## 运行

```bash
cd "d:/专题2：VOC分析/voc-data-product/app/dashboard/frontend"
npm install
npm run dev
```

默认访问：

- `http://localhost:3000/` 首页
- `http://localhost:3000/explore` 前台数据浏览
- `http://localhost:3000/admin` 后台管理看板

请配置环境变量（可选）：

```bash
NEXT_PUBLIC_API_BASE=http://127.0.0.1:8000
```
